# ♟️ Chess Engine From Scratch

![Python](https://img.shields.io/badge/Python-3.x-blue.svg)
![Status](https://img.shields.io/badge/Status-In%20Development-orange)
![License](https://img.shields.io/badge/License-MIT-green)

A **chess engine built completely from scratch in Python** without using external chess libraries.

The goal of this project is to deeply understand how chess engines work by implementing everything manually — from board representation and move generation to search algorithms and eventually a **self-improving AI through reinforcement learning**.

This project also serves as a learning journey into:

- Algorithms
- Game theory
- Search algorithms
- Artificial intelligence

---

# ✨ Features (Planned)

- Full chess rules implementation
- Legal move generation
- Command line chess interface
- Human vs Human gameplay
- Human vs Bot gameplay
- Chess engine using **Minimax**
- **Alpha-Beta pruning** optimization
- Graphical interface
- Self-learning AI through **self-play**
